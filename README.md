## What is this?

A light weight photo-server designed to run on single-board computer (raspberry pi and the likes)

### Running the server

```sh
cargo run -- path_to_config.toml
```

### Running tests

```sh
cargo test
```

### Viewing coverage report

```sh
./coverage-report.sh
```
